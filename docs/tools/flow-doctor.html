📋 How to get your API credentials

  1. 1.Go to Homey API Clients (log in with the same Athom account you use for your Homey).
  2. 2.Click "Add client".
  3. 3.Give it any name (e.g. "Flow Doctor").
  4. 4.Paste the Redirect URL shown below.
  5. 5.Select scopes: homey.flow.readonly · homey.app.readonly · homey.device.readonly · homey.zone.readonly
  6. 6.Click "Create" and copy the Client ID and Client Secret below.
